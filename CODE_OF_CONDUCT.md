# Code de conduite

## Notre engagement

Nous nous engageons à maintenir un environnement respectueux et professionnel.

## Règles

- Respecter les autres contributeurs.
- Utiliser un langage approprié.
- Accepter les critiques constructives.
- Signaler tout comportement inapproprié.

## Application

Les mainteneurs du projet sont responsables de l'application de ce code de conduite.
